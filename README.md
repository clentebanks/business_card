# Tarjeta digital — Clent Ebanks

Tarjeta de presentación digital adaptada a la marca personal de Clent Ebanks.

## Identidad

- Posicionamiento: Tecnología clara para personas reales.
- Mensaje: De militar a crear y enseñar tecnología.
- Proyecto principal: WinHacks.
- Segundo producto: Ebanks Interpreter Studio, en validación.
- Empresa futura: Ebanks Labs.

## ✨ Features
- Fully responsive design using Bootstrap 5
- Mobile-first layout
- Customizable contact info, services, and social links
- Simple HTML, CSS, and JavaScript
- Clean, minimalist design for a professional appearance

## 📁 What's Included
- `index.html` – Main HTML file
- `style.css` – Custom styles
- `scripts.js` – Optional JavaScript for interactivity
- `assets/` – Images, icons, and other resources
- `README.md` – Documentation
- `LICENSE.md` – License and usage rights

## 🛠️ How to Use
1. Download the ZIP and unzip it.
2. Open `index.html` in your browser or code editor.
3. Edit the content to match your personal or business details.
4. Upload to your web host (e.g., Netlify, GitHub Pages, Vercel).

## 📝 Customization
Replace your name, photo, contact information, and links in the HTML file. Modify colors and layout in `style.css`.
